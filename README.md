# nbext
Noteblock extender pack

This pack allows extra octaves in minecraft. it can be used by placing wood underneath a noteblock to make it one octave lower, stone for two octaves lower, and sand for two octaves higher. For one octave higher, hit the noteblock twelve times.

**Older versions are available [here](https://www.curseforge.com/minecraft/texture-packs/nbext)!**
